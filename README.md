# ExpenseAI

A full stack expense tracker built with React and Appwrite, with AI-powered financial insights using Google Gemini. Users can track their expenses, set a monthly budget, visualise spending patterns through charts, and get personalised saving advice from AI.

Live demo: [expense-ai-gilt.vercel.app](https://expense-ai-gilt.vercel.app)

---

## What it does

- Sign up, log in, and log out with secure authentication
- Add, view, and delete transactions with category, amount, and date
- Dashboard with summary cards — total spent, budget left, average monthly, top category, transaction count
- Set a monthly budget and track how much is remaining
- Spending breakdown with area chart, pie chart, and stacked bar chart
- AI-powered insights — summary, key insights, and saving tips generated by Google Gemini
- Responsive design with a bottom navigation bar on mobile
- Transactions persist across sessions via Appwrite database

## Environment variables

Create a `.env` file in the root of the project:

```
VITE_APPWRITE_URL=https://cloud.appwrite.io/v1
VITE_APPWRITE_PROJECT_ID=your_project_id
VITE_APPWRITE_DATABASE_ID=your_database_id
VITE_APPWRITE_COLLECTION_ID=your_collection_id
VITE_GEMINI_API_KEY=your_gemini_api_key
```

Never commit this file. It is already in `.gitignore`. See `.env.sample` for the required variable names.

## Running locally

Clone the repo and install dependencies

```bash
git clone https://github.com/MAR-ARIF/ExpenseAI-Expense-tracker-App.git
cd ExpenseAI-Expense-tracker-App
npm install
```

Add your `.env` file with your own Appwrite and Gemini credentials, then run:

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

## AI insights

The insights feature sends your transaction data to Google Gemini and asks it to return a financial summary, key spending insights, and saving tips as structured JSON. The response is split into three separate colour-coded sections in the UI.

The Gemini API free tier is used — no billing required. Rate limits apply (250 requests per day on the free tier).
